# ENEI 2027 - Faro Tech Hub

Apresentação estratégica de candidatura para o Encontro Nacional de Estudantes de Informática 2027 em Faro.

## Estrutura do Projeto

```
enei2027/
├── index.html          # Estrutura principal das 12 slides
├── css/
│   └── style.css       # Estilos completos separados
└── README.md           # Documentação
```

## Slides

1. **Capa** - ENEI 2027 Faro no Epicentro
2. **O Que é o ENEI?** - Apresentação do evento
3. **Magnitude e Impacto** - Estatísticas principais
4. **Pilares Estratégicos** - Conhecimento, Networking, Cidadania
5. **Organização & Comissão** - Equipa organizadora
6. **Infraestruturas Elite** - Locais do evento
7. **Plano de Execução** - Atividades programadas
8. **Projeção de Impacto Local** - Dados económicos
9. **O Talento Atendido** - Distribuição por área
10. **Faro: O Próximo Hub Tecnológico** - Crescimento previsto
11. **Roadmap de Execução** - Cronograma
12. **Contacto** - Informações de contacto

## Tecnologias

- HTML5 semântico
- CSS3 com variáveis e grid/flexbox
- Font Awesome para ícones
- Google Fonts (Urbanist, JetBrains Mono)

## Como Usar

Abrir o ficheiro `index.html` diretamente no browser ou servir via servidor web.
